# LiveEdit 🌐

> **Turn any website into your personal sandbox.**

> **LiveEdit** is a lightweight, high-utility browser extension that gives you total control over a website's client-side appearance. Whether you need to mock up a quick change, swap an image for a presentation, or delete distracting elements, LiveEdit makes the entire DOM interactive with zero technical friction.

---

## 🚀 Features
- **Native Text Editing:** Toggles `designMode` to make every headline, paragraph, and button text editable like a Word document.
- **Image Swapper:** `Alt + Click` any image to instantly replace its source URL.
- **Element Killer:** `Shift + Click` any element (ads, sidebars, popups) to remove it from the page instantly.
- **OLED Stealth HUD:** A clean, professional status overlay that shows active hotkeys.
- **Instant Dismiss:** Press `Esc` or click the 'X' to hide the HUD while keeping Edit Mode active.
- **Non-Destructive:** All changes are client-side only. A simple refresh restores the original site.

---

## 🛠 Tech Stack
- **Vanilla JavaScript:** Fast, dependency-free execution.
- **Chrome Extension API:** Manifest V3 compliant.
- **DOM Injection:** Injects a secure overlay that stays on top of any website architecture.

---

## 🛠 Support & Updates
- **Manual Updates Only:** Stable versioning. This extension does not track you or auto-update.
- **Refresh Note:** Always refresh your active tab after installation or update to initialize the sandbox environment.
- **DesignMode Limitation:** Some highly complex web apps (like Google Docs) have custom text engines that may resist standard `designMode` editing.

---

## ⚙️ Installation
1. Download the source code.
2. Navigate to `chrome://extensions/` in your browser.
3. Enable **Developer Mode** in the top-right corner.
4. Click **Load unpacked** and select the folder named **LiveEdit**.
